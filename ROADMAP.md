# Roadmap

This roadmap keeps future additions focused on causal inference for experimentation and decision-making. Items here should become README resources only when there is a strong, stable, technically useful source to link.

## Near Term

- Add more high-quality resources on variance reduction in online experiments, including CUPAC and modern regression-adjustment practice.
- Add sequential testing, alpha spending, always-valid inference, and experiment-monitoring resources.
- Add cluster-randomized, geo, marketplace, and split-plot experiment designs.
- Add resources on interference, spillovers, network experiments, and marketplace feedback loops.
- Add missing-data, attrition, censoring, and survivorship-bias resources.
- Add starter paths for product analysts, economists, ML researchers, and epidemiologists.

## Medium Term

- Add causal inference resources for recommender systems, ranking, ads, and ML product loops.
- Add more healthcare, education, labor, and public-policy case studies.
- Add resources on transportability, external validity, and treatment-effect generalization.
- Add practical examples of sensitivity analysis and negative-control workflows.
- Add resources on experiment platform governance, metric review, and launch decision processes.

## Maintenance

- Keep the README selective rather than comprehensive.
- Prefer official docs, author pages, publisher pages, and maintained repositories.
- Revisit archived or dormant projects periodically.
- Use GitHub issues for proposed additions that need more review.
- Treat link-check and awesome-lint failures as maintenance work, not noise.

## Review Standard

Every new resource should help readers answer at least one of these:

- What is the causal question?
- What is the estimand?
- What identifies the counterfactual?
- What assumptions are required?
- What diagnostics or falsification checks are available?
- What are the important failure modes?
